            \vskip3.5em
            \begin{center}
                $L_{CIoU} = 1 - IoU +R_{CIoU}$
                \vskip0.5em
                $R_{CIoU} = \frac{\rho^2(b,b^{gt})}{c^2} + \alpha v$
                \vskip0.5em
                $\alpha = \frac{v}{(1 - IoU) + v}$
                \vskip0.5em
                $v = \frac{4}{\pi}(\arctan{\frac{w^{gt}}{h^{gt}}} - \arctan{\frac{w}{h}})^2$
                \vskip2em
                $\textup{Точность} = \frac{TP}{TP + FP}$
            \end{center}

            \vskip1.5em
            $TP$ --- число истинно положительных распознаваний\\
            $FP$ --- число ложноположительных распознаваний

    \frametitle{\LARGE Цель и задачи}
    \vskip3.5em

    \textbf{Цель} --- разработка метода распознавания надводных объектов с аэрофотоснимков с использованием нейронных сетей.

    \textbf{Задачи}:
\begin{itemize}
    \item Проанализировать нейросетевые методы распознавания объектов
    \item Разработать соотвествующий метод распознавания
    \item Разработать программное обеспечение, реализующее метод распознавания надводных объектов
    \item Оценить результаты работы метода в зависимости от различных параметров системы
\end{itemize}
